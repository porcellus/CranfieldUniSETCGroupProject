Class KeyExchange

java.lang.Object
  com.jcraft.jsch.KeyExchange
Direct Known Subclasses:
DHG1, DHG14, DHGEX
public abstract class KeyExchange
extends Object

Usually not to be used by applications. Abstract base class for key exchange algorithms. The concrete implementation will be chosen based on the configuration and on the negotiation between client and server.

See Also:
RFC 4253, 7. Key Exchange, RFC 4253, 8. Diffie-Hellman Key Exchange

init

public abstract void init(Session session,
                          byte[] V_S,
                          byte[] V_C,
                          byte[] I_S,
                          byte[] I_C)
                   throws Exception
Initializes the key exchange object. The parameters here seem to be specific to the Diffie-Hellman Key exchange specified by RFC 4253, other Key Exchange mechanisms might need other parameters.

Parameters:
session - the session object.
V_S - the server's identification string sent before negotiation
V_C - the client's identification string sent before negotiation
I_S - the server's complete SSH_MSG_KEXINIT message.
I_C - the server's complete SSH_MSG_KEXINIT message.
Throws:
Exception
See Also:
RFC 4253, 7. Key Exchange, RFC 4253, 8. Diffie-Hellman Key Exchange
