Class DHG14

java.lang.Object
  com.jcraft.jsch.KeyExchange
      com.jcraft.jsch.DHG14

public class DHG14
extends KeyExchange

Usually not to be used by applications. Implements the key exchange method diffie-hellman-group1-sha1, using the Oakley Group 14. For the actual cryptographic calculations we delegate to an implementation of DH.

See Also:
RFC 4253, section 8. Diffie-Hellman Key Exchange, RFC 3526, section 3. 2048-bit MODP Group
