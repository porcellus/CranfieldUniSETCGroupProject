Interface ServerSocketFactory

public interface ServerSocketFactory

A factory for ServerSockets. This interface works similarly to ServerSocketFactory, but does not depend on the 1.4 Java version.

An application may want to implement this interface to have control over the server socket creation for local port forwardings, for example to configure the server socket (or the Sockets it creates) before use.

See Also:
SocketFactory, Session.setPortForwardingL(String, int, String, int, ServerSocketFactory)

Creates a ServerSocket.

Parameters:
port - the port to listen on.
backlog - the number of not-yet accepted connections which may be waiting at the same time before new ones will be rejected.
bindAddr - the local network interface the socket will be listening on. If null, listen on all local addresses.
Throws:
IOException - if some network error occured, like the port was already in use.
