Class KeyPair

java.lang.Object
  com.jcraft.jsch.KeyPair
Direct Known Subclasses:
KeyPairDSA, KeyPairRSA
public abstract class KeyPair
extends Object

Base class for a pair of public and private key. This class is used internally by the library and not necessary for normal use by the application.

An application may use this class to generate new keys with genKeyPair, and then write the KeyPair to files, which then might be used with JSch.addIdentity.

Another possible use would be to load a key pair in order to change the passphrase.
