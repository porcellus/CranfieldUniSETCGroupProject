com.jcraft.jsch
Interface UserInfo

public interface UserInfo

Allows user interaction. The application can provide an implementation of this interface to the Session to allow for feedback to the user and retrieving information (e.g. passwords, passphrases or a confirmation) from the user.

If an object of this interface also implements UIKeyboardInteractive, it can also be used for keyboard-interactive authentication as described in RFC 4256.

Most of the examples include an implementation of this interface based on Swings JOptionPane.

See Also:
